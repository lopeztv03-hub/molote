## Installation

### Requirements
- Python 3.11+ (3.11 for undetected_chromedriver, 3.14+ for standard Selenium)
- pip (Python package manager)

### Setup

**Option 1: Standard Selenium (Python 3.14+)**
```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements_std.txt
```

**Option 2: Undetected Chrome (Python 3.11)**
```powershell
py -3.11 -m venv .venv311
.venv311\Scripts\Activate.ps1
pip install -r requirements_311.txt
```
